## 使用说明（当前正式无金币版本）
- 广告触发型：提示 / 额外预览 / 生命补给 / 洗牌 / 首页广告补给
- 库存消耗型：找一对 / 护盾 / 冻结
- 签到 / 每日挑战 / 章节奖励 / 特殊牌应发放道具或时间类收益，不应再出现金币奖励或金币扣费
- 若进入 Round 4 局内功能级回归，请同时参考：
  - `ROUND4_REGRESSION_PLAN.md`
  - `TEST_REPORT_ROUND4.md`
  - `TEST_REPORT_ROUND4_TEMPLATE.md`
